# 📈 Cartera AR — Análisis de Portafolios

Aplicación web para análisis de portafolios de inversión con foco en el mercado argentino (CEDEARs) e internacional.

## Funcionalidades

- **📊 Análisis de Cartera** — Markowitz, frontera eficiente, fundamentales
- **📈 Análisis Técnico** — RSI, MA 9/21, Squeeze Momentum, ADX, Order Blocks
- **🇦🇷 CEDEARs** — Valor implícito vs precio ARS con CCL en tiempo real
- **💼 Mi Cartera** — Multi-cartera, P&L en tiempo real, movimientos
- **📰 Info de Mercado** — Noticias, ratings de analistas, métricas ETF

## Tecnologías

- Python 3.12+
- Streamlit
- yfinance
- Plotly
- Pandas / NumPy / SciPy

## Ejecutar localmente

```bash
pip install -r requirements.txt
cd scripts
streamlit run app.py
```
